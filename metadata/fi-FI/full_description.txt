BiglyBT is an <b>ad-free</b>, fully featured open source torrent client and remote control app, optimized for phone, tablet, Chromebook, and Android TV.

★ Swarm Merging for <b>faster torrent downloading</b> and to <b>repair torrents</b> without seeds
★ v2 Torrent Support (BEP 52)
★ Tag and <b>organize</b> your torrents
★ Sort and grouping for <b>large lists</b> of torrents
★ Select which files are downloaded before starting the torrent
★ Sort and filter files in torrent
★ View progress, peers, and stats of torrents
★ Tilaa RSS-syötteitä
★ Use Swarm Discoveries to <b>find related</b> torrents
★ All the basic torrent features: Mainline DHT, Vuze DHT, UPnP, uTP, PEX, UDP Tracker, Encryption
★ Download to SD Card via SAF (Storage Access Framework)
★ Control other BiglyBT, and Transmission RPC compatible desktop torrent clients
★ Can access remotely via LAN
★ Tumma vai vaalea teema
★ Ainoa täystoiminen Leanback UI -käyttöliittymää käyttävä torrent-sovellus <b>Android TV:lle</b>.
