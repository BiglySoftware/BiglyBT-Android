BiglyBT è un client torrent open source completamente funzionale, <b>privo di pubblicità</b> ed app di controllo remoto, ottimizzata per telefoni, tablet, Chromebook ed Android TV.  Comparaci alle migliori app "torrent" a <a href="http://goo.gl/h26KnS">Comparazione Client Torrent Android</a> ( http://goo.gl/h26KnS )

★ Swarm Merging for <b>faster torrent downloading</b> and to <b>repair torrents</b> without seeds
★ Tagga ed <b>organizza</b> i tuoi torrent
★ Ordina e raggruppa per <b>grande elenco</b> di torrent
★ Seleziona quali file sono scaricati prima di avviare il torrent
★ Ordina e filtra file in torrent
★ Vedi progresso, pari e statistiche dei torrent
★ Iscriviti ai feed RSS
★ Usa Scoperte Sciame per <b>trovare i relativi</b> torrent
★ Tutte le funziono torrent base: Mainline DHT, Vuze DHT, UPnP, uTP, PEX, Tracciatore UDP, Decrittografia
★ Control other BiglyBT, Vuze, and Transmission RPC compatible desktop torrent clients
★ Can access remotely via LAN
★ Dark or Light Theme
★ The only full torrent app for <b>Android TV</b> using the Leanback UI.
